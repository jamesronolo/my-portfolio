### Update Restaurant Information
Edit the following in `index.html`:
- Restaurant name and tagline (Hero section)
- About Us text (About section)
- Menu items and prices (Menu section)
- Address, phone, and hours (Contact section)
- Social media links and contact buttons

### Add Social Media Links


### Enable Online Ordering


### Update Google Map
Replace the iframe `src` with your restaurant's location:
1. Go to [Google Maps Embed](https://www.google.com/maps)
2. Search for your restaurant address
3. Click "Share" > "Embed a map"
4. Copy the iframe code and replace the existing one
